# 📚 영업점 베스트셀러 차트 관리 시스템

## 📖 프로젝트 개요
영업점별, 파트별, 조별 베스트셀러 리스트를 체계적으로 관리하고 매장 비치용으로 손쉽게 출력하기 위한 웹 애플리케이션입니다. 
최신 주차와 이전 주차의 베스트셀러 엑셀 데이터를 업로드하면 순위 변동 내역을 자동으로 비교하여 화면에 렌더링하며, 인쇄에 최적화된 UI를 통해 오프라인 매장의 업무 효율을 극대화합니다.

## ✨ 주요 기능
- **자동 순위 비교**: 금주 및 전주 베스트셀러 엑셀 데이터(.xlsx, .csv 등)를 업로드하여 순위 상승, 하락, 유지, 신규 진입을 자동 계산
- **세밀한 영업점 필터링**: 전국 영업점명, 파트, 조 단위로 카테고리를 설정하여 맞춤형 리스트 조회 가능
- **인쇄 최적화**: 상단 바 및 불필요한 UI를 숨기고 오직 도서 목록만 깔끔하게 출력할 수 있는 전용 모드 제공
- **도서 표지 확인**: 직관적인 도서 표지 모달을 통해 비치되어야 할 실물 도서의 썸네일을 빠르게 확인
- **클라우드 데이터 동기화**: Supabase를 활용한 매장 간 설정 및 데이터 상태 연동

## 🛠 기술 스택
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Radix UI (shadcn/ui)
- **Backend/DB**: Supabase (Edge Functions, Deno)
